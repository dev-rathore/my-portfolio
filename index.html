My Projects

Pixel Store

Pixel Store is an E-commerce store for buying products like camera, gimbals, tripods, drones built using ReactJS. It also has authentication and database that are implemented using Firebase authentication and firestore. For state management in ReactJS, React Context API is used. And routing is done using React Router DOM. This store is deployed on Netlify.

Rent & Ride

Rent & Ride is a vehicle rental website that has feature for renting different types of vehicles e.g. four wheeler, two wheeler for a short period of time. It is built using technologies like HTML, CSS, JavaScript, Bootstrap, SASS, NodeJS, ExpressJS, Passport.js, MongoDB. This website is deployed on Heroku.

Weather Web App

Weather Web App is a web app that let's you know the weather conditions of any particular city. This is built using NodeJS, ExpressJS, EJS, Weather API, HTML, CSS. This web app is deployed on Heroku Server.
